## Editing Notes

- Keep cuts fast: 1.5s to 2.5s per scene
- Use the product UI as the primary footage
- Show real token pages when possible
- Avoid promising fully automatic external LP deployment unless the live product supports it
- Use `@Tempo` in first-touch clips to anchor the chain identity
- Use `$TEB` only where the fee-policy message is the core subject
